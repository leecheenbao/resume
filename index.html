企業 EAP 管理系統

專案簡介

此專案為嘉澎塑膠打造之內部員工互動與績效管理平台，結合 RPG 冒險遊戲概念與 EAP（Employee Assistance Program）制度，透過任務與積分機制推廣 ESG 永續理念，強化員工參與度與企業文化建構，並結合績效評估與升遷制度，形成完整的人才激勵循環。

系統特色

  • 遊戲化設計：以 RPG 冒險任務為核心，員工可透過完成 ESG 相關行動（如環保、志工、社區活動）獲得「小星點」、「月亮幣」積分
  • 積分激勵機制：積分除可兌換公司福利外，亦納入年終考績、升遷評估依據，鼓勵員工持續參與正向行為
  • 後台管理平台：提供任務申請、審核與積分管理，支援多角色操作與審核流程
  • 多企業推廣：採用多租戶架構（Multi-Tenant），可擴展至其他企業使用，協助推動 ESG 企業文化生態圈

技術亮點

  • 設計並實現完整的企業 EAP 管理系統架構，採用前後端分離開發模式
  • 使用 Node.js 和 Express 框架開發 RESTful API，確保系統擴展性
  • 實作 JWT 身份驗證機制，整合多角色權限管理系統
  • 設計多租戶架構（Multi-Tenant），支援橫向擴充與跨企業推廣
  • 整合遊戲化機制與人事制度，實現積分激勵與績效評估系統
  • 使用 MySQL 資料庫管理複雜的任務、積分與用戶數據關係

健康管理系統 Health Management System

專案簡介

結合藥品管理與病患健康追蹤的全方位健康管理平台，透過 QR Code 掃描、用藥提醒與衛教資訊，協助病患有效管理個人健康，同時提供後台給管理員維護藥品與衛教資料。

前台功能

  • 病患資料建立與會員登入：首次掃描藥盒 QR Code 即自動建立病患資料，並可選擇使用 Google 第三方登入進行會員註冊與身分綁定
  • 用藥數據分析與提醒：依據不同藥品種類與劑量，系統自動建立個人化用藥提醒，追蹤病患的服藥狀況與用藥紀錄
  • 回診提醒功能：病患可設定回診時間，系統於到期前自動發送通知提醒
  • 衛教文宣中心：提供最新的衛教文章與健康知識，協助病患建立正確的用藥與健康觀念

後台功能

  • 藥品資訊維護：後台提供藥品資料的新增、修改、下架等功能，確保前台用藥建議的準確性
  • 衛教文章管理：管理者可透過後台建立與維護衛教內容，支援標籤分類與發布狀態控制

技術亮點

  • 設計並實現完整的健康管理系統架構，採用前後端分離開發模式
  • 使用 Node.js 和 Express 框架開發 RESTful API，確保系統擴展性
  • 實作 JWT 身份驗證機制，整合 Google OAuth 第三方登入
  • 整合 QR Code 掃描功能與用藥提醒系統，提升用戶體驗

HR內部選課系統

專案簡介

為企業內部使用的線上選課與課程管理平台，旨在讓員工可自主推薦或報名公司課程，並由人資部門進行課程審核、公告與點數管理。系統採用前後台分離架構，支援身分驗證、課程報名流程與點數制管理，協助企業提升內部教育訓練流程效率。

前台功能

  • 登入機制：僅限內部授權員工登入，透過 Email + OTP 驗證機制確保安全。登入 Email 與後台員工資料比對
  • 課程選擇與報名：前台顯示最多 10 門開放課程，員工可直接報名；若點數不足則無法報名
  • 課程推薦：員工可推薦課程給人資審核，包含講者、主題與內容等基本資訊，流程簡單類似活動表單
  • Outlook 整合：報名成功後，系統可將課程自動加入 Microsoft Outlook 行事曆
  • 個人中心：員工可查看剩餘點數、報名紀錄、提案狀態與課程報名情況

後台功能

  • 員工管理：管理者可新增、移除員工，並可進行點數轉移（例如移除員工時自動轉移其點數）
  • 點數管理：管理者可手動發放點數、調整員工點數餘額；報名未成課或取消報名時，點數自動或人工返還
  • 員工資料匯入：支援批次匯入員工資料（員工編號、姓名、Email），作為登入驗證依據
  • 課程管理：管理者可新增、編輯、刪除課程，設計課程內容（標題、描述、時間、圖片、報名人數等），並決定是否上架
  • 課程狀態控制：可設定課程為「報名課程」或「投票課程」，靈活應對不同活動形式
  • 顯示限制：同時最多顯示 10 門上架課程，確保介面簡潔明瞭

技術亮點

  • 設計並實現完整的選課系統架構，採用前後端分離開發模式
  • 使用 Node.js 和 Express 框架開發 RESTful API，確保系統擴展性
  • 實作 JWT 身份驗證機制，提升系統安全性
  • 整合 GCP Cloud Storage 處理多媒體內容，優化資源管理
  • 使用 Github Actions 建立 CI/CD 自動化部署流程，提高開發效率
Node.js Express MySQL Docker GCP
